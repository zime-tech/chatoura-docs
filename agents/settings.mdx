title: "Agent Settings"
description: "Configure language, tone, and persona for your AI agent"
---

Fine-tune how your AI agent communicates with users through language, tone, and persona settings.

## Language Settings

### Preferred Language

Set the primary language for responses:

- **English** - Default
- **Arabic** - Full Arabic support
- **Spanish**
- **French**
- **German**
- **Auto-detect** - Responds in the user's language

### Dialect

For Arabic and English, choose a dialect:

| Language | Dialects |
|----------|----------|
| Arabic | Egyptian, Formal Arabic, Gulf, Levantine |
| English | Formal, Friendly |

<Note>
Dialect affects vocabulary, expressions, and cultural references in responses.
</Note>

## Tone Settings

Control the communication style:

| Tone | Best For |
|------|----------|
| **Friendly** | General customer support, casual interactions |
| **Professional** | B2B, enterprise, legal, financial |
| **Casual** | Social media, young audiences |
| **Formal** | Government, academic, official communications |
| **Enthusiastic** | Sales, marketing, product launches |

## Persona Settings

Define your agent's personality and behavior:

### Display Name
The name shown to users in the chat widget. Can differ from the internal agent name.

### System Instructions
Custom instructions that guide agent behavior:

```
You are a helpful customer support agent for [Company Name].
Always be polite and professional.
If you don't know an answer, offer to connect the user with a human agent.
Never discuss competitor products.
```

### Response Guidelines
- Maximum response length
- Topics to avoid
- Required information to collect
- Escalation triggers

## Applying Settings

1. Go to **Agents** > select your agent > **Settings**
2. Configure language, tone, and persona
3. Click **Save**
4. Test changes in the **Playground**

<Warning>
Changes take effect immediately for new conversations. Existing conversations continue with previous settings.
</Warning>
