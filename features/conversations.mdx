The Chat History page displays all chat sessions across your channels. Review message history, monitor AI interactions, and manage session status.

## Chat History Dashboard

Go to **Chat History** to see:

- All chat sessions across channels
- Channel source (Web, Facebook, Instagram, Playground)
- Session status (Active or Paused)
- Last message preview
- Session timestamps

## Session Status

| Status | Meaning |
|--------|---------|
| Active | AI agent is responding to messages |
| Paused | AI responses are temporarily disabled |

## Viewing a Session

Click any session to see:

- Full message history
- Timestamps for each message
- AI and user message indicators
- Channel type badge
- Session status

## Filtering Sessions

### By AI Agent

Use the dropdown at the top to filter sessions by a specific AI agent or view all agents.

### By Search

Search sessions using the search field. Enter at least 2 characters to search across session data.

## Managing Session Status

You can pause or activate AI responses for any session:

1. Select the session
2. Click **Pause Session** or **Activate Session** in the header
3. Status updates immediately

<Note>
When a session is paused, the AI agent stops responding to new messages. Activate the session to resume AI responses.
</Note>

## Channel Types

Sessions are labeled by their source channel:

| Channel | Description |
|---------|-------------|
| Web | Messages from the web widget |
| Facebook | Messages from Facebook Messenger |
| Instagram | Messages from Instagram DMs |
| Playground | Test messages from the agent playground |

## Real-Time Updates

Chat history updates in real-time:

- New messages appear instantly
- Status changes reflect immediately
- No need to refresh the page

## Session Display Names

Sessions are identified by:

- **Facebook/Instagram** - Username when available, otherwise numbered (e.g., "Instagram 1")
- **Web** - Numbered by creation order (e.g., "Web 1", "Web 2")
- **Playground** - Numbered by creation order

## Best Practices

- **Review regularly** - Check chat history to monitor AI performance
- **Pause when needed** - Pause sessions that require special attention
- **Learn from interactions** - Identify knowledge gaps from conversation patterns
- **Update training** - Use insights to improve AI responses

## Troubleshooting

<AccordionGroup>
  <Accordion title="Sessions not appearing">
    - Check channel connection status
    - Verify agent is active
    - Refresh the page
    - Check the agent filter isn't hiding sessions
  </Accordion>
  <Accordion title="Messages delayed">
    - Check internet connection
    - Verify channel webhooks are active
    - Some channels have inherent delays
  </Accordion>
  <Accordion title="Search not working">
    - Enter at least 2 characters
    - Try different search terms
    - Clear filters and try again
  </Accordion>
