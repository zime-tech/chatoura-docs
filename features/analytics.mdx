Analytics help you understand how your AI agent is performing. Track conversations, measure engagement, and identify opportunities to improve.

## Dashboard Overview

Go to **Analytics** to see key metrics:

- Total conversations
- Messages sent/received
- Lead capture rate
- Escalation rate
- Credit usage

## Key Metrics

### Conversation Metrics

| Metric | Description |
|--------|-------------|
| Total conversations | All conversations in period |
| Active conversations | Currently ongoing |
| Avg. messages per conversation | Engagement depth |
| Avg. conversation duration | Time from start to resolution |

### Performance Metrics

| Metric | Description |
|--------|-------------|
| Response rate | % of messages that received responses |
| Escalation rate | % of conversations escalated |
| Resolution rate | % of conversations resolved by AI |
| Lead capture rate | % of conversations with lead captured |

## Using Analytics to Improve

### Low Resolution Rate

If AI isn't resolving conversations:

1. Review unanswered questions
2. Add Q&A pairs for common issues
3. Update knowledge base
4. Adjust agent instructions

### High Escalation Rate

If too many conversations escalate:

1. Check escalation triggers
2. Identify common escalation causes
3. Train AI for those scenarios
4. Adjust sensitivity settings

### Low Lead Capture

If not capturing enough leads:

1. Review lead capture settings
2. Check agent prompting for contact info
3. Consider adding lead form
4. Test conversation flow

## Best Practices

- **Check weekly** - Regular review catches issues early
- **Compare periods** - Track improvement over time
- **Act on insights** - Use data to drive changes
- **Set goals** - Target metrics to improve
- **Share with team** - Align on performance
