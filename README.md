# Chatoura Documentation

Official documentation for Chatoura - the multi-channel AI chatbot platform.

## Documentation Structure

- **Getting Started** - Introduction and quickstart guide
- **AI Agents** - Creating and configuring AI agents
- **Knowledge Base** - Training agents with documents, websites, and Q&A
- **Integrations** - Web widget, Facebook, Instagram
- **Features** - Leads, conversations, escalations, analytics
- **Billing** - Plans and credits
- **Developer Guide** - Widget embedding and API usage
- **API Reference** - Complete API documentation

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview documentation locally:

```bash
npm i -g mint
```

Run at the root of the documentation:

```bash
mint dev
```

View at `http://localhost:3000`.

## Publishing

Changes pushed to the main branch are automatically deployed via Mintlify.
